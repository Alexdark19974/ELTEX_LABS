# Shared memory (POSIX)
## Задание 1:
### Написать программу, которая создаёт разделяемую область памяти и пишет строку "Hello, World!"
### Вторая программа подсоединяет разделяемую память и считывает оттуда строку на экран. 
____

#### Сборка проекта:

'~$ make all'

#### Запуск сервера:

'~$ make shm_writer'

#### Запуск клиента:

'~$ make shm_reader':

#### Удаление проекта:

'~$ make clean'
