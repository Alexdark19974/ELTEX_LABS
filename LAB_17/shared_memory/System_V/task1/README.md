# Shared memory (SYSTEM V)
## Задание 1:
### Написать программу, которая использует разделяемую память, записывает в неё строку "Hello, World!", а второй процесс выводит эту строку.
____

#### Сборка проекта:

'~$ make all'

#### Запуск проекта:

'~$ make shm1_writer'

'~$ make shm1_reader'

#### Удаление проекта:

'~$ make clean'
