# Статический калькулятор
## Задание:
### Написать статический калькулятор используя статическую библиотеку.
### Библиотека должна включать все базовые операции: сложение, вычитание, умножение, деление.
____

#### Сборка проекта:

'~$ make all'

'~$ make test_calc'
