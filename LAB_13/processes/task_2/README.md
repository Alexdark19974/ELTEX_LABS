# Порождение иерархии процессов
## Задание:
### Написать программу, которая создаёт иерархию процессов, где родитель порождает 2х потомков. Первый потомок порождает ещё 2 потомка, а второй порождает всего 1 потомка.
____

#### Сборка проекта:

'~$ make all'

#### Запуск проекта:

'~$ make fork_2'

#### Удаление проекта:

'~$ make clean'
____
