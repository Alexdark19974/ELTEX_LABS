# Mini-bash
## Задание:
### Написать небольшой интерпретатор комманд "bash", который эмулирует  ввод данных по следующему типу:
'$ ls -al'
### bash должен порождать новые процессы путём вызова fork() и функций из семейства exec()

#### Сборка проекта:

'~$ make all'

#### Запуск проекта:

'~$ make bash'

#### Удаление проекта:

'~$ make clean'
