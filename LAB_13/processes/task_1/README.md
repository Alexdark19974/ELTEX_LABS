# Порождение процесса-потомка
## Задание: 
### Написать программу, которая порождает дочерний процесс, а затем выдёргивает его PID, PPID, и выводит на экран. То же самое и для процесса-родителя.
____

#### Сборка проекта:

'~$ make all'

#### Запуск проекта:

'~$ make fork'

#### Удаление проекта:

'~$ make clean'
____
