# Primitive MC (Midnight Commander)
## Задание:
### Написать подобие mc, который бы реализовывал переход из папки в папку, в обоих направлениях, а также имел бы возможность переключаться между 2 окнами.
____

#### Сборка проекта:

'~$ make all'

#### Запуск проекта:

'~$ make mc'

#### Удаление проекта

'~$ make clean'
____
